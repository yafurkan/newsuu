# Badges

Bu klasör rozet (badge) görsellerini içerir.

## Kullanım
- PNG formatında rozetler (şeffaf arka plan)
- Boyut: 128x128 px
- Dosya adlandırma: `badge_[kategori]_[seviye].png`

## Kategoriler
- **Günlük:** `badge_daily_[1-30].png`
- **Haftalık:** `badge_weekly_[1-52].png`
- **Özel:** `badge_special_[isim].png`

## Örnekler
- `badge_daily_7.png` (7 gün serisi)
- `badge_weekly_4.png` (4 hafta serisi)
- `badge_special_champion.png` (Şampiyon rozeti)
